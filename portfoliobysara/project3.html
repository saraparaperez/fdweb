about:

Helvetica is a typographic project focused on exploring the expressive potential of the typeface as the main visual element. The design emphasizes clarity, structure, and visual hierarchy through typographic composition, using scale, weight, and spacing as key tools for experimentation.
The project takes the form of an A4 leaflet folded into four panels using an accordion layout. This format creates a sequential reading experience that reinforces the rational and systematic nature of the typeface while allowing playful typographic interactions within a controlled grid.
On the reverse side, the leaflet unfolds into a poster where typography interacts with photography. While both elements coexist, typography remains the dominant focus, with imagery serving as a complementary element. This dual-format approach highlights the versatility of Helvetica across editorial and poster-based applications.
